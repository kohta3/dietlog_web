プライバシーポリシー

dietlog

最終更新日：2025年3月6日

dietlog（以下「本アプリ」）は、体重管理とダイエットのモチベーション向上をサポートするアプリです。 本プライバシーポリシーは、本アプリが収集・利用・保存する情報について説明します。

1. 収集する情報

本アプリは以下の情報を収集・保存します。

1.1 アカウント情報

  • メールアドレス
  • パスワード（暗号化されて保存）

1.2 プロフィール情報

  • ユーザー名（表示名）
  • ユーザーID
  • 目標体重
  • 身長
  • 性別
  • 目標タイプ（減量・維持・増量）
  • プロフィール画像

1.3 体組成・体重データ

  • 体重
  • BMI
  • 筋肉率
  • 体脂肪率
  • ウエスト周囲径
  • 測定日時

1.4 ソーシャル機能に関する情報

  • フォロー・フォロワー関係
  • 友達リクエスト
  • タイムラインへの投稿内容（体重変化、目標達成状況など）

1.5 自動収集される情報

  • デバイス情報（OS、アプリバージョンなど）
  • アプリの利用状況（画面遷移、機能の使用頻度など）
  • クラッシュ・エラー情報

2. 情報の利用目的

  • アカウントの作成・認証・管理
  • 体重・体組成データの記録・グラフ表示
  • 目標達成の進捗管理
  • タイムライン・友達機能の提供
  • プッシュ通知の送信
  • アプリの改善・不具合の修正
  • 利用状況の分析によるサービス品質の向上

3. 利用する第三者サービス

本アプリは以下の第三者サービスを利用しており、これらのサービスがデータを処理する場合があります。

3.1 Google Firebase（Google LLC）

  • Firebase Authentication：認証
  • Cloud Firestore：データベース（プロフィール、体重データ、タイムライン等）
  • Firebase Storage：プロフィール画像の保存
  • Firebase Analytics：利用状況の分析
  • Firebase Crashlytics：クラッシュレポート
  • Firebase Cloud Messaging：プッシュ通知

Firebase のプライバシーポリシー： https://firebase.google.com/support/privacy

3.2 Microsoft Clarity（Microsoft Corporation）

利用状況の分析に Microsoft Clarity を使用しています。Clarity のプライバシーポリシー： https://privacy.microsoft.com/privacystatement

4. データの保存場所とセキュリティ

  • データは Google Firebase のサーバーに保存されます。
  • 通信は SSL/TLS により暗号化されています。
  • パスワードは Firebase Authentication により安全に管理されます。
  • 機密情報のローカル保存には Flutter Secure Storage を使用しています。

5. データの保持期間

アカウント削除のリクエストがあった場合、または法令に基づく場合を除き、 サービス提供に必要な期間、データを保持します。アカウント削除時には、関連するデータを削除します。

6. お客様の権利

お客様は以下の権利を有します。

  • 収集されたデータへのアクセス
  • データの訂正・削除の要求
  • アカウントの削除

これらのご要望は、お問い合わせ先までご連絡ください。

7. 子どもに関する方針

本アプリは13歳未満の子どもを対象としていません。13歳未満の子どもの個人情報を意図的に収集することはありません。

8. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更することがあります。重要な変更がある場合は、 アプリ内でのお知らせや本ページの更新日を変更するなどして通知します。
